# Group

Knowledge and work that apply to **every** sub-company in Team System. If
something is true only for one company, it belongs in `companies/<name>/` instead
— not here. Like every scope, the group splits into two halves plus scope-level
meta files.

| Path | Contents |
|---|---|
| [`content/`](content/) | The group brain: [`knowledge/`](content/knowledge/), [`decisions/`](content/decisions/), [`brainstorms/`](content/brainstorms/). |
| [`code/`](code/) | Shared group-level code, if any (one project; stay flexible). |
| [`journal.md`](journal.md) | Append-only log of what each session changed at the group level. |

See the root [`CLAUDE.md`](../CLAUDE.md) for the lifecycle and the continuous loop.
