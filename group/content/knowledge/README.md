# Group knowledge

The settled, current truth shared across **all** companies. Each doc has
frontmatter (`title`, `status`, `owner`, `updated`, `tags`) — see
[`templates/knowledge-doc.md`](../../../templates/knowledge-doc.md).

| Subfolder | What belongs here |
|---|---|
| [`engineering/`](engineering/) | Shared technical standards: stack, conventions, security, runbooks. |
| [`comms/`](comms/) | Shared brand & communications: voice, messaging, visual basics. |
| [`operations/`](operations/) | How the group runs: tooling, ways of working, policies. |

Keep these current. When a decision changes reality, update the affected doc and
mark superseded content.
