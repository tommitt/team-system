# Dott. Comm. creatives & ads

Creative deliverables for Dott. Comm. — ad concepts, social videos, and the
prompts used to generate them.

- `video-prompts/` — ready-to-use text-to-video prompts for ads to produce later.
  Each file is one self-contained prompt (master prompt + beat sheet + production
  notes) so it can be dropped straight into a video model.
